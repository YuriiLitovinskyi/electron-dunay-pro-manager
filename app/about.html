About App

This application can export some statistic info from Dunay-PRO Database.

How to use:

  • Enter database connection options
  • Press button "Connect To Database"
  • Press "Show Dun128 Conn Stat" to find when devices sent message Restart for the first time
    • Select Begin and End dates
    • Press button "Apply"
    • If there were any devises found - press "Export To Excel" button
    • Choose directory and press "Save"
  • Press "Show Employees Stat" to export data about employees
    • Press button "Export Statistics to Excel"
    • Choose directory and press "Save"
  • Press "Disconnect from Database" button
